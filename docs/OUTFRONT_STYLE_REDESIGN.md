# OUTFRONT Media Style Redesign

## Overview

Redesigned Movrr homepage sections using **OUTFRONT Media** as the primary design reference, with JCDecaux, Lamar Advertising, Clear Channel Outdoor, and Ströer as secondary references. The design now positions Movrr as an enterprise-level urban media network operator.

**Visual Category**: OUTFRONT Media × JCDecaux × Clear Channel Outdoor

---

## Design Principles Applied

### 1. **Enterprise Authority**
- Strong 12-column grid systems
- Large, confident typography (text-5xl to text-9xl)
- Industrial, minimal aesthetic
- High-contrast design
- System-driven layouts

### 2. **OUTFRONT-Style Elements**
- Full-width containers (max-w-[1400px])
- Strong border system (border-2, border-t-2)
- Industrial typography (font-black, tracking-tight)
- Structured content blocks
- Professional photography presentation

### 3. **Urban Media Network Aesthetic**
- Network status panels
- Coverage metrics
- Operational data presentation
- Campaign showcase layouts
- Process documentation style

## Typography System

### OUTFRONT-Style Scale

| Element | Size | Weight | Tracking |
|---------|------|--------|----------|
| Hero | `6xl → 7xl → 8xl → 9xl` | `font-black` | `tracking-[-0.02em]` |
| Section Headers | `5xl → 6xl → 7xl` | `font-black` | `tracking-tight` |
| Subheadings | `xl → 2xl` | Regular | `leading-relaxed` |
| Body | `lg → xl` | Regular | `leading-relaxed` |
| Labels | `xs` | `font-bold` | `tracking-[0.15em]` |
| CTAs | `base` | `font-bold` | `tracking-[0.1em]` |

### Typography Patterns

- **Headlines**: Massive scale, `font-black`, tight tracking
- **Labels**: Uppercase, `tracking-[0.15em]`, border-bottom accent
- **Body**: Large (`text-lg` to `text-xl`), relaxed leading
- **CTAs**: Uppercase, `tracking-[0.1em]`, bold

---

## Layout System

### Grid System
- **Container**: `max-w-[1400px]` (OUTFRONT standard)
- **Padding**: `px-4 sm:px-6 lg:px-8 xl:px-12` (full-width feel)
- **12-column grid**: Used throughout for structured layouts
- **Gap system**: `gap-8 lg:gap-12` for generous spacing

### Border System
- **Section separators**: `border-t-2 border-border`
- **Content blocks**: `border-2 border-border`
- **Accents**: `border-b-2 border-primary` for labels
- **Grid separators**: `border-r-2`, `border-b-2` for structured grids

### Spacing System
- **Section padding**: `py-32 md:py-40` (OUTFRONT standard)
- **Content spacing**: `space-y-8` to `space-y-10`
- **Header spacing**: `mb-20` for section headers

---

## Component Patterns

### Buttons
- **Style**: `rounded-none`, `border-2`
- **Primary**: `border-2 border-primary bg-primary`
- **Secondary**: `border-2 border-foreground`
- **Typography**: Uppercase, `tracking-[0.1em]`, `font-bold`
- **Size**: `h-14` to `h-16` for prominence

### Cards/Blocks
- **Style**: `border-2 border-border`, no rounded corners
- **Background**: `bg-card` or `bg-background`
- **Padding**: `p-10 lg:p-12` for generous spacing
- **Hover**: Subtle `hover:bg-card/30`

### Metrics Panels
- **Style**: `border-2 border-border bg-card`
- **Layout**: Structured grid with labels
- **Typography**: Large numbers (`text-2xl` to `text-4xl`), `font-black`
- **Labels**: Uppercase, `tracking-[0.15em]`, small

---

## Color Usage

### Primary Colors
- **Primary Green**: Used for accents, borders, CTAs
- **Foreground**: Deep black for maximum contrast
- **Background**: Pure white
- **Borders**: `border-border` for structure

### Contrast
- **High contrast**: Black text on white
- **Border system**: Strong 2px borders throughout
- **Accents**: Primary color for emphasis
